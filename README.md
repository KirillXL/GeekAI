# GeekAI

GeekAI — это веб-сайт с встроенным искусственным интеллектом, который рекомендует пользователю книги, фильмы и игры на основе его предпочтений. Приложение реализовано на Flask и содержит три основные страницы:

- **/ (auth)** — страница аутентификации (шаблон `auth.html`)
- **/chat** — чат-страница (шаблон `chat.html`)
- **/admin** — админ-панель (шаблон `admin.html`)

## Функции (в разработке)

- Рекомендации книг, фильмов и игр на основе предпочтений пользователя
- Чат-бот с поддержкой искусственного интеллекта
- Персонализированные советы и диалоги
- Админ-панель для управления контентом

## Структура проекта

```
app.py            # Основной Flask-приложение
main.py           # (Не используется напрямую в app.py)
static/           # Статические файлы (JS, CSS, изображения)
  admin.js
  chat.js
  door.png
  styles.css
templates/        # HTML-шаблоны
  admin.html
  auth.html
  chat.html
```

## Запуск проекта

1. Установите зависимости:

```bash
pip install flask
```

2. Запустите приложение:

```bash
python app.py
```

3. Откройте в браузере: [http://127.0.0.1:5000/](http://127.0.0.1:5000/)

## Описание страниц

- **auth.html** — форма входа или регистрации
- **chat.html** — интерфейс чата с поддержкой искусственного интеллекта (ИИ). На этой странице реализован чат-бот, который может отвечать на вопросы пользователей и вести диалог в реальном времени.
- **admin.html** — страница администратора

## Статические файлы

- **styles.css** — стили для страниц
- **admin.js, chat.js** — JS-логика для соответствующих страниц
- **door.png** — изображение (например, для оформления)
